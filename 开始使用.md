# 🎬 视频搜索助手 - 快速开始

## 这是什么？

一个智能助手，帮你在 **YouTube** 和 **Instagram** 上找到最热门的视频！

### 功能
- 🔍 搜索热门视频（播放量 > 20万）
- 🤖 AI 智能筛选（只推荐最相关的）
- ⚡ 速度快（10-15秒）
- 💰 成本低（每次约 $0.03）

---

## 三步开始使用

### 第1步：获取 API Keys（免费）

#### Gemini API Key
1. 打开：https://ai.google.dev/
2. 登录 Google 账号
3. 点击 "Get API Key"
4. 复制 Key

#### YouTube API Key
1. 打开：https://console.cloud.google.com/
2. 创建项目
3. 启用 "YouTube Data API v3"
4. 创建 API Key
5. 复制 Key

### 第2步：安装和配置

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 创建配置文件
cp env_template.txt .env

# 3. 编辑 .env 文件，填入你的 API Keys
# 使用任何文本编辑器打开 .env
```

在 `.env` 文件中填入：
```
GEMINI_API_KEY=你的gemini_key
YOUTUBE_API_KEY=你的youtube_key
```

### 第3步：开始搜索！

```bash
python main.py "AI编程工具"
```

## 使用示例

### 命令行使用
```bash
# 直接搜索
python main.py "健身教程"
python main.py "美食制作"
python main.py "旅游vlog"

# 交互式
python main.py
# 然后输入主题
```

### Python 代码使用
```python
from video_agent import VideoSearchAgent

# 创建助手
agent = VideoSearchAgent()

# 搜索视频
results = agent.search("Python教程", top_n=10)

# 查看结果
for video in results:
    print(f"{video['title']} - {video['views']:,} 播放")
```

## 常见问题

### Q: 必须有 Instagram 账号吗？
**A**: 不必须。不登录也能用，但登录效果更好。

### Q: API Key 要钱吗？
**A**: 
- Gemini: 按使用付费（很便宜）
- YouTube: 免费配额够用

### Q: 搜索结果准确吗？
**A**: 使用 Google Gemini AI 分析，准确度很高！

### Q: 可以搜索中文内容吗？
**A**: 可以！支持任何语言。

### Q: 数据会保存吗？
**A**: 会缓存2小时，之后自动删除。你可以手动导出。

## 调整设置

编辑 `video_agent/config.py`：

```python
MIN_VIEWS = 200000        # 最小播放量（可调低）
MAX_DAYS_AGO = 60         # 最近N天内（可调大）
TOP_N_RESULTS = 10        # 返回数量（可调整）
CACHE_EXPIRY_HOURS = 2    # 缓存时间（小时）
```
